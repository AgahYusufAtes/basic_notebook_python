# 📔 Terminal-Based Smart Notepad with Python Loops and Filters

This project is a terminal-based **Smart Notepad** application developed using the core logic filters (`if-elif-else`) and infinite loop (`while True`) mechanisms of the Python programming language.

It is one of the first comprehensive projects I designed to improve my software architecture and algorithmic thinking skills during my preparation for a university career in Aerospace Engineering.

---

## 🚀 Features
* **Memory Management:** Stores entered notes safely in the RAM while the program is running.
* **Smart Filtering:** Prevents attempting to display a note before one has actually been entered, warning the user.
* **Input Security (Bug Protection):** Detects invalid keypresses other than 1, 2, or 3 (including accidental blank entries), preventing the system from crashing and safely resetting the loop.
* **Secure Exit:** Terminates the loop and program safely using the `break` command.

---

## 🛠️ Technologies Used
* **Language:** Python 3
* **Structures:** `while` loops, `if-elif-else` conditional statements, `input()` function, string manipulations.

---

## 💻 How to Run
Simply run the script in your local Python environment or IDE and follow the terminal menu prompts (1, 2, 3).
